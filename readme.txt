Integrates your WP site with your PennyCloak account.  

== Description ==

PennyCloak.com is a tool for cloaking your redirected links from web crawlers as well as from affiliate programs.  You can get an account and learn more about the service at www.pennycloak.com.

This plugin allows you to easily configure your WP site to work with your PennyCloak account to cloak the posts and pages on your site.


== Installation ==

1. You can download and install PennyCloak using the built in WordPress plugin installer. But If you download PenyCloak manually, make sure it is uploaded to "/wp-content/plugins/pennycloak/".

2. Once it is installed, click on "Activate" in the plugin menu.

3. Go to Settings --> PennyCloak  to configure the plugin.  Get your unique ID from your Control Panel at pennycloak.com (http://panel.pennycloak.com/) and paste the ID into the box provided.

4. When you create a post or page, you will notice a box that says "Cloak this Post".  If you want this post to work with PennyCloak, click the box.  Only posts that have been activated with the checkbox will be compatible with PennyCloak.

5. Go to your Setup page at PennyCloak (http://panel.pennycloak.com/tab/setup) to configure the redirect for this page.


== Frequently Asked Questions ==

= Do I need a PennyCloak account for this to work? =

Yes.  The plugin is free but in order for it to work yo must sign up for an account at www.pennycloak.com

= Can I use this plugin on multiple WP sites and manage them all via a single Pennycloak account =

Yes.  You can manage as many sites and domains as you want with a single PennyCloak account.
